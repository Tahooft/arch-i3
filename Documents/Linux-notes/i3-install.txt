# arch
Archlinux install script

For use after the first setup with Archinstall. 
Set in archinstall:
    • hostname
    • user account
    • audio
    • additional packages – rxvt-unicode, tmux, vim, thunar
    • networkmanager
    • evt optinal repisotories
	

    1. Test if we are working on a laptop, VM or desktop pc
    2. Config sudo and wheel for user
    3. Optimizing makepkg
    4. Installing essential packages
    5. Installing themes, fonts and icons
    6. Replacing dotfiles with my own
    7. Installing extra packages (docker, ddev, ufw ..)
