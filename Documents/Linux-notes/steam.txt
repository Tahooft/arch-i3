OpenSSL 1.0 setup
Some Steam games are built against OpenSSL 1.0. (FS#53618)
Install lib32-openssl-1.0 and run the game with LD_LIBRARY_PATH=/usr/lib/openssl-1.0.

First, install ciopfs (how to do that, will depend on your distribution and I can not cover it here).

cd ~/.local/share
mkdir -p ciopfs-storage/steamlib2
mkdir ciopfs
ciopfs $HOME/.local/share/ciopfs-storage $HOME/.local/share/ciopfs  (is mounting de case-insensitive directory )
